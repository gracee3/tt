# doctor

Use this skill when the task is to diagnose a failure, localize a bug, or explain why a path is broken.

## What this skill does

- gathers symptoms and reproducer details
- narrows the failure boundary
- separates evidence from speculation
- recommends the next smallest diagnostic step

## What this skill does not do

- broad cleanup by default
- feature expansion
- silent root-cause claims without evidence
