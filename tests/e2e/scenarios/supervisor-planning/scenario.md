# Scenario: Supervisor Planning Mode

## Goal

Verify that the supervisor can draft a plan-like proposal, review it, and keep planning artifacts separate from execution until the operator approves them.

## Steps

1. Create a workstream with an ambiguous objective.
2. Create a work unit that needs a short plan.
3. Generate a proposal that contains a multi-step plan.
4. Review the proposal text.
5. Approve it only after the plan is inspectable.

## Expected Result

- A planning artifact exists before execution.
- The proposal can be reviewed and approved explicitly.
