# clean

Use this skill when the task is to clean up stale files, reduce clutter, or improve repository hygiene.

## What this skill does

- removes or consolidates stale artifacts
- keeps the repo readable
- preserves useful history while trimming noise
- treats cleanup as a deliberate action, not a side effect

## What this skill does not do

- delete useful evidence
- rewrite semantics for style alone
- over-clean during an active investigation
