# agent

Use this skill when the task is to spawn, coordinate, or maintain subagents.

## What this skill does

- assigns narrow, bounded tasks to other lanes
- keeps ownership boundaries explicit
- integrates returned findings without duplicating work
- preserves the orchestration trail

## What this skill does not do

- scatter unrelated work
- hide responsibilities between agents
- spawn work without a clear next question
