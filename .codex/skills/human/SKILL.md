# human

Use this skill when the task is to coordinate with the human operator and keep requirements clear.

## What this skill does

- asks the smallest useful clarification
- keeps the operator oriented
- makes constraints and open decisions visible
- avoids over-questioning

## What this skill does not do

- take over implementation
- bury the operator in broad choices
- ignore the current thread state
