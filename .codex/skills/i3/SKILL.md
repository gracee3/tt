# i3

Use this skill when the task involves i3 window management or desktop coordination.

## What this skill does

- manages window placement, focus, and workspace intent
- coordinates operator-visible surfaces
- keeps UI arrangement changes deliberate

## What this skill does not do

- application logic changes
- process lifecycle outside window-manager coordination
- arbitrary desktop automation without a clear operator goal
